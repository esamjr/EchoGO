# EchoGO

Simple Rest API with Echo & MySQL

## Usage

$ go run server.go

## Endpoints

| Method | Route       |
| ------ | ----------- |
| GET    | /pegawai    |
| POST   | /pegawai    |
| PUT    | /pegawai    |
| DELETE | /pegawai    |
| POST   | /login(*username & password)      |
| POST   | /register(*username & password)   |
